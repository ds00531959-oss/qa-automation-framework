# 🚀 QA Automation Framework

This is an End-to-End QA Automation Framework built using **Selenium** and **PyTest** following the **Page Object Model (POM)** design pattern.

---

## 🔧 Tech Stack
- Python
- Selenium WebDriver
- PyTest
- Page Object Model (POM)

---

## 📌 Features
- ✅ Login Automation (Valid & Invalid scenarios)
- ✅ Add to Cart functionality
- ✅ Checkout flow automation
- ✅ Structured using POM
- ✅ Reusable and maintainable code
